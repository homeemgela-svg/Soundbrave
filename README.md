# 🎙️ Soundbrave

Soundbrave is an AI-powered meeting and lecture recording application that records audio, transcribes speech into text, summarizes discussions, and helps users organize notes from classes, meetings, and interviews.

## Features

- 🎤 One-tap audio recording
- 📝 AI-powered transcription
- 👥 Speaker identification
- 📋 Automatic summaries
- 🔍 Transcript search
- 📂 Folder organization (Classes, Meetings, Projects)
- 🌍 Multi-language support (English, Hausa, Pidgin, French)
- 📄 Export to PDF, DOCX, and TXT
- ☁️ Secure cloud storage
- 🌙 Dark mode (planned)
- 📅 Google Calendar integration (planned)
- 🤖 AI quiz generation (planned)
- 📖 Technical glossary mode (planned)

## Technologies

- HTML5
- CSS3
- JavaScript
- React (future web version)
- React Native (future mobile version)
- Whisper AI (speech-to-text)
- GPT API (summaries and notes)

## Installation

1. Download or clone the repository.

```bash
git clone https://github.com/yourusername/soundbrave.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
